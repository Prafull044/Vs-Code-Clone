# VS Code Clone

A Visual Studio Code (VS Code) landing page clone built using HTML5 and CSS3. This project was created to enhance frontend development skills by recreating a real-world website interface, focusing on layout design, navigation, styling, and user interface implementation.

## Features

* VS Code-inspired UI Design
* Navigation Bar
* Hero Section
* Feature Cards
* Footer Section
* Clean and Structured Code
* Pure HTML & CSS Implementation

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was developed as a frontend practice project to improve HTML and CSS skills, understand website structure, and gain experience in building modern user interfaces from scratch.
